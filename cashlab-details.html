CashLab - Personal Finance & Pawn Broking Platform

CashLab is a comprehensive personal finance platform offering cash loans, pawn broking services, and an online shop. The platform provides trusted cash solutions when customers need it most, with both online application and in-store services available.

Key Features:

  • Personal finance loan application system
  • SACC loans ($300-$2,000) and MACC loans ($2,050-$5,000)
  • Pawn broking services with XRF testing
  • Online shop with luxury brands
  • Store locator (Midland & Geraldton)
  • User account management
  • Secure payment processing
  • Warranty and refund system
  • Responsive design for all devices
  • E-commerce functionality

Technologies Used:

HTML5, CSS3, JavaScript, PHP, Laravel, MySQL, Bootstrap, E-commerce Integration, Payment Gateway, Responsive Design, User Authentication
